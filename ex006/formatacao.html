Principais Formatações

Negrito / Destaque

Para colocar um texto em negrito, use a tag <strong> (semântico)

Itálico / Ênfase

Para colocar um texto em itálico use a tag <em> (semântico)

Marcador

Tambem podemos marcar um texto usando a tag <mark>

Texto grande/ pequeno

Para fazer um texto ficar grande use a formatação do CSS, mas se desejar um texto menor pode usar a tag <small>

Texto Deletado

Podemos marcar um texto como deletado utilizando a tag <del>

Texto Inserido

Podemos marcar um texto como inserido usando a tag <ins>

Texto sobrescrito

Para inserir um texto sobrescrito utilize a tag <sup>

Texto subscrito

Para finalizar se precisar de um texto subscrito use a tag <sub>
